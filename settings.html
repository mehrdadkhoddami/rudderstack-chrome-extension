localStorage Key Patterns

Define which localStorage keys the extension should read. Each rule is a startsWith prefix (required) and an optional endsWith suffix. Built-in rules can be toggled; custom rules can be added and removed freely.

Built-in rules

rudder_* … *.batchQueue Keys starting with rudder_ and ending with .batchQueue — default RudderStack SDK format
queue.* Keys starting with queue. — used by snappfood and similar setups
